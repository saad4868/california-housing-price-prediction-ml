# 🏠 California Housing Price Prediction (Machine Learning Project)

This project uses Machine Learning to predict house prices using the **California Housing Dataset**.

---

## 📊 Dataset

We use the built-in dataset from sklearn:

California Housing Dataset  
Features include:

- Median income
- House age
- Average rooms
- Average bedrooms
- Population
- Latitude
- Longitude

Target:
- House price (median value)

---

## 🤖 Model Used

- Linear Regression

---

## ⚙️ Steps in Project

1. Load dataset
2. Split data into training and testing
3. Train Linear Regression model
4. Make predictions
5. Evaluate model performance

---

## 📦 Libraries Used

- numpy
- pandas
- scikit-learn

---

## 🧪 Model Evaluation

We use:

- Mean Squared Error (MSE)
- R² Score
